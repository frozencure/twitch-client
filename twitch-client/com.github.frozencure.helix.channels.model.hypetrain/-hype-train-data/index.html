An object that holds the data about a hype train event.

Parameters

cooldownEndTime

expiresAt
Expiration timestamp of this hype train event.
goal
The goal value of reaching the next level of the hype train.
id
The ID of the this hype train event.
lastContribution
The most recent contribution.
level
The highest level of the hype train (on a scale of 1-5) reached.
startedAt
Timestamp of when this hype train started.
topContributions
An array of top contribution objects, one object for each type.
total
The total score so far towards reaching the next level of the hype train.
